컴파일 및 실행 실습

자바의 기본 실행단위는 .class


  • 파일명.java → javac 파일명.java → [파일명.class] → java 파일명


💻 실습1 : 기초


Step01. 코드를 HelloWorldApp.java 로 저장

/**

The HelloWorldApp class implements an application that
simply prints "Hello World!" to standard output.*/
class HelloWorldApp { // 클래스 선언
	// 코드 영역
	public static void main(String[] args) { // 프로젝트 진입점 메인 메소드 HelloWorldApp.main()
		System.out.println("Hello World!"); // Display the string. -> 현재 시스템.표준출력장치.줄바꿈포함프린트("문자열");
	}
}


Step02. 컴파일

javac HelloWorldApp.java   # 컴파일한 후 HelloWorldApp.class 생성 확인
java HelloWorldApp         # 실행

💻 실습3 : 패키지에 넣어서 컴파일

Test.java를 com.test.Test.java로 패키지에 넣어서 컴파일 실행


Step01. 코드 작성

package com.test;
public class Test{}


Step02. 현재 디렉토리에 있는 Test.java를 com.test 하위로 이동

mv Test.java com\test
move Test.java com\test # window10


Step03. 컴파일

javac -d . com/test/Test.java
  • -d . 옵션 : .(현재 디렉토리)를 기준으로 패키지 구조를 유지하며 .class 파일을 생성


Step04 실행

java com.test.Test

접근 제한자 & 파일 작성법


접근 제한자

  • public : 공개
  • protected : 상속시 공개
  • default : 같은 패키지 공개
  • private : 비공개

클래스

클래스 작성 방법

  • 하나의 파일의 여러 개의 클래스 선언 가능

    X.java 

      class X{ }
  class Y{ }
  class Z{ }


  • 접근제한자 public 클래스는 하나의 파일에 하나만 선언한 후, public 클래스 이름으로 저장

    X.java 

      // error
  public class X02{ }
  public class Y02{ }
  public class Z02{ }

  // ✔
  public class X02{ }
  class Y02{ }
  class Z02{ }

    Z.java 

      class X{ }
  class Y{ }
  public class Z{ }

               


  • 클래스 안에 클래스를 중첩으로 사용 가능

    X1.java     X1$Y1.class 

      class X1 {
      class Y1 { }
  }


Case | 여러 개의 단일 클래스

  • 각각의 클래스가 생성됨
      class X{ }
  class Y{ }
  class Z{ }

  • 실행방법 : 각 클래스로 실행
      java X
  java Y
  java Z


**Case중첩 클래스**
public class AA{
    class AA_inner{}
    class AA_inner01{}
}


식별자 규칙 및 선언


접근제한자 [public, protected, default, private] class UserName{
		명령 코드
		Nested | Field | Constr | Method 
		// Nested | Field : 변수역할
		// Constr | Method : 메소드 역할
	}


UserName 종류

  • 프로젝트, 모듈, 패키지, 클래스
  • 클래스 = 필드(변수) + 메소드


UserName 규칙

  • 영문자, 숫자, _ , $ 로만 이름 짓는다
  • 대소문자 구분
  • 공백 X
  • 숫자로 시작 X
  • 예약어 불가


클래스 표기법

  • 클래스명 → 파스칼 표기법 ; 첫글자 대문자
    • e.g. HelloWorldApp, Test
  • 메소드명 → 카멜 표기법 ; 첫글자 소문자
    • e.g. A.hap(), A.getHap()
  • 변수명 → 카멜 표기법
    • e.g. String userName=”홍길동” userAddress
  • 상수 (static field) → 대문자 + 언더스코어
    • e.g. MAX_VALUE
    • e.g. public static final double PI → 공유한다 정적영역에서 단일명 더블값의 PI라는 이름으로
